Mental Health Support Api

This API has been designed to allow web developers to implement features which will return information which is related to mental health, and the support which is in place in the UK.


Disclaimer: this API has been developed as part of a learning experience and it is not monitored or updated, therefore I do not take any responsibility for any negative outcomes of using the services, and if someone needs mental health support, it is always best to seek professional medical support.


To search by condition use the following URL as an API and replace 'params' with the relevant condition (current search options are: suicide, bereavement, bipolar, depression, addiction, OCD and panic): 'https://mh-support-api.herokuapp.com/api/support/byspec/[param]'.
Param is replaced by the condition .
